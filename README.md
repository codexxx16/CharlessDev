# CharlessDev

Personal portfolio of Charless Netumbare — Full Stack Developer and AI Engineer from Masvingo, Zimbabwe.

## Tech Stack

- **Framework**: [Next.js 15](https://nextjs.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Database**: [Supabase](https://supabase.com/)
- **Content**: [MDX](https://mdxjs.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Deployment**: [Vercel](https://vercel.com/)

## Features

- **Modern UI/UX**: Clean, responsive design with dark mode support.
- **Dynamic Dashboard**: Real-time stats from GitHub and Supabase.
- **Guestbook**: Anonymous message board powered by Supabase.
- **Blog**: MDX-based blog with syntax highlighting.
- **Projects**: Showcase of latest work and client projects.

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/codexxx16/CharlessDev.git
   ```
2. Install dependencies:
   ```bash
   bun install
   ```
3. Set up environment variables:
   Copy `.env.example` to `.env.local` and fill in your Supabase and GitHub credentials.
4. Run the development server:
   ```bash
   bun dev
   ```
