## Getting Started

### Prerequisites

- Node.js v18.17+
- npm or pnpm

### Installation

```bash
# Clone the repository
git clone https://github.com/codexxx16/CharlessDev.git

# Install dependencies
npm install

# Run development server
npm run dev
```

The portfolio will be available at `http://localhost:3000`.

## Project Structure

- `/src/app` - Next.js app directory with pages and layouts
- `/src/app/resources` - Configuration and content files
- `/src/components` - Reusable React components
- `/src/once-ui` - Once UI design system components
- `/public/images` - Static images and assets

## Customization

### Update Personal Information

Edit `/src/app/resources/content.js` to update:
- Personal details (name, role, location)
- Social media links
- About section
- Work experience
- Technical skills
- Projects

### Update Configuration

Edit `/src/app/resources/config.js` to:
- Change base URL
- Enable/disable routes
- Customize theme colors and styles
- Adjust visual effects

### Add Projects

Create new MDX files in `/src/app/work/projects/` with the following structure:

```mdx
---
title: "Project Title"
publishedAt: "2024-01-01"
summary: "Brief description"
images: ["/images/projects/project-name/cover.jpg"]
tag: "Technology"
team: []
---

Project content here...
```
