This site — my personal portfolio, blog, and live dashboard. Built on a fork of nelsonlai.dev and fully customized with my own identity, content, and backend.

The backend uses Supabase for blog views, likes, guestbook entries, and comments — replacing the original Redis + Drizzle + Better Auth stack with a simpler, more maintainable architecture.

Features include MDX blog with syntax highlighting, interactive globe showing my location in Zimbabwe, live GitHub stats, and a personal dashboard tracking coding hours, gaming time, and content metrics.
