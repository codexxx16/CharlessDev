ZimPay is a production-grade payment gateway that unifies Zimbabwe's fragmented mobile money ecosystem — EcoCash, OneMoney, and Zimswitch — behind a single, developer-friendly API.

The platform handles real-time transaction processing, webhook verification, and provides a merchant dashboard with live analytics. Built with security-first principles including Row Level Security, webhook signature verification, and full audit logging.

Key features include multi-provider support, real-time transaction monitoring via Supabase subscriptions, idempotent payment initiation, and a clean REST API with TypeScript SDK.
