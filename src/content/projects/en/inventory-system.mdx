RetailTrack is a real-time inventory management system built for a multi-branch retail chain in Zimbabwe. The system tracks stock levels across locations, generates automated reorder alerts, and provides management dashboards with sales analytics.

Built with Supabase real-time subscriptions for live stock updates, Row Level Security for branch-level data isolation, and a responsive Next.js frontend that works on both desktop and mobile.

The system processes thousands of transactions daily and has reduced stockout incidents by 60% for the client.
