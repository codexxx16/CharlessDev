An AI-powered customer service platform built for a local Zimbabwean retail business. The system uses RAG (Retrieval-Augmented Generation) to answer customer queries using the company's product catalog, FAQs, and policy documents.

Built with a FastAPI backend, pgvector for semantic search, and OpenAI's GPT-4o for response generation. The system handles hundreds of queries daily with sub-2-second response times.

Key features include streaming responses, function calling for real-time inventory checks, conversation history, and a human handoff system for complex queries.
